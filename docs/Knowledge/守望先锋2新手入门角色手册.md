## 1. 游戏基础介绍

### 1.1 游戏模式

**《守望先锋2》**是一款6v6团队射击游戏，玩家需要与其他队友合作完成任务。

| 模式 | 简介 |
|------|------|
| **快速对战** | 标准6v6对战，适合练习 |
| **竞技模式** | 有排名的团队对抗 |
| **街机模式** | 各种特殊规则的小游戏 |
| **自定义游戏** | 与朋友练习特定内容 |

### 1.2 胜利条件

- **攻防作战**: 进攻方需将 payload（运载目标）运送到检查点
- **占领要点**: 占领并守住目标区域
- **攻击/防御**: 在限定时间内占领目标点

### 1.3 核心玩法

1. **团队配合** - 单打独斗很难获胜
2. **角色切换** - 根据战况更换英雄
3. **技能协调** - 团队技能同步释放
4. **位置意识** - 保持有利站位

## 2. 角色分类详解

游戏中的英雄分为三大类型：

| 类型 | 图标颜色 | 职责 | 数量 |
|------|----------|------|------|
| **坦克型 (Tank)** | 蓝色 | 保护队友、冲锋陷阵 | 4-5名 |
| **输出型 (Damage)** | 红色 | 造成伤害、击杀敌人 | 10+名 |
| **支援型 (Support)** | 绿色 | 治疗队友、提供增益 | 4-5名 |

### 2.1 团队配置建议

**标准阵容**: 2名坦克 + 2名输出 + 2名支援

- **爆头训练**: 爆头伤害是身体伤害的2倍
- **预判移动**: 敌人移动方向前方开枪
- **蹲下射击**: 可以提高近距离命中率

#### 🛡️ 生存技巧
- **保持移动**: 静止目标容易被击中
- **利用掩体**: 躲避敌方火力
- **注意血量**: 及时寻找治疗或掩体
- **团队行动**: 不要落单

#### 👥 团队配合
- **及时沟通**: 报点敌方位置
- **技能协调**: 配合队友释放大招
- **角色切换**: 根据敌方阵容更换英雄
- **保护治疗**: 坦克优先保护支援

## 7. 地图机制介绍

### 7.1 攻防作战（Push）

**地图**: 新加坡、蒙特利尔、罗马

**机制**:
- 机器人将沿着路径移动
- 双方需要占领并推进机器人
- 到达终点或推更远的一方获胜

### 7.2 占领要点（Control）

**地图**: 尼泊尔、奥斯蒙德、伊利欧

**机制**:
- 占领并保持目标区域
- 保持100%占领度获胜
- 平局时进入加时赛

### 7.3 攻击/防御（Escort/Hybrid）

**地图**: 好望角、纽卡斯尔、里约热内卢

**机制**:
- 攻击方护送运载目标到达终点
- 防守方阻止运载进度
- 在时间耗尽前到达检查点则胜利
