## 3. 员工行为规范

### 3.1 基本行为准则

1. **遵纪守法**: 严格遵守国家法律法规和公司规章制度
2. **爱岗敬业**: 认真履行岗位职责，积极主动完成工作
3. **团结协作**: 尊重同事，乐于助人，密切配合
4. **诚实守信**: 说实话，办实事，守信用
5. **勤俭节约**: 爱护公物，合理使用公司资源

### 3.2 职业道德规范

#### 3.2.1 忠诚义务
- 维护公司利益，不从事损害公司利益的活动
- 保守公司商业秘密，未经授权不得泄露公司信息
- 不得利用职务之便谋取私利

#### 3.2.2 公正公平
- 在工作中公平对待每一位同事和客户
- 不得因个人好恶、亲疏远近而影响决策
- 避免利益冲突，主动申报潜在利益冲突

#### 3.2.3 专业精神
- 持续学习，提升专业能力
- 以专业的态度对待工作和客户
- 不断追求工作质量和效率的提升

### 3.3 日常行为规范

| 场景 | 规范要求 |
|------|----------|
| **出勤** | 准时上下班，不得迟到早退 |
| **着装** | 保持整洁得体，符合岗位要求 |
| **办公** | 保持安静，不得大声喧哗 |
| **会议** | 准时参加，将手机调至静音 |
| **用餐** | 有序排队，爱护餐厅卫生 |
| **网络** | 合理使用网络资源，不得浏览非法网站 |

### 3.4 禁止行为

- 🚫 贪污受贿、挪用公款
- 🚫 泄露公司商业秘密
- 🚫 组织或参与非法集会、游行
- 🚫 在工作场所酗酒、赌博
- 🚫 侮辱、诽谤同事或客户
- 🚫 故意损害公司或他人财产
- 🚫 违反保密协议相关规定

## 4. 入职指南

### 4.1 入职流程

```
发送Offer → 背景调查 → 体检 → 签订劳动合同 → 办理入职手续 → 参加入职培训 → 正式上岗
```

### 4.2 入职材料清单

| 序号 | 材料名称 | 份数 | 备注 |
|------|----------|------|------|
| 1 | 身份证原件及复印件 | 各2份 | 正反面复印 |
| 2 | 学历证书原件及复印件 | 各1份 | 
| 3 | 学位证书原件及复印件 | 各1份 | 本科及以上提供 |
| 4 | 离职证明 | 1份 | 原单位开具 |
| 5 | 社保转移材料 | 1份 | 异地参保员工 |
| 6 | 照片 | 4张 | 白底一寸 |
| 7 | 体检报告 | 1份 | 二级甲等及以上医院 |
| 8 | 银行卡复印件 | 1份 | 用于发放工资 |

### 4.3 劳动合同

#### 合同类型
- **固定期限劳动合同**: 适用于一般岗位，合同期限1-3年
- **无固定期限劳动合同**: 适用于核心岗位或连续工作满10年的员工
- **以完成一定工作任务为期限的劳动合同**: 适用于项目制岗位

#### 试用期规定
- 试用期不超过6个月
- 试用期工资不低于正式工资的80%
- 试用期缴纳五险一金
- 试用期转正评估由直属上级和HR共同完成

### 4.4 新员工入职培训

| 培训项目 | 培训时间 | 培训内容 |
|----------|----------|----------|
| 公司概况培训 | 入职第1天上午 | 公司历史、组织架构、企业文化 |
| 制度流程培训 | 入职第1天下午 | 规章制度、行为规范 |
| 业务介绍培训 | 入职第2天 | 各业务线介绍、产品知识 |
| 部门业务培训 | 入职第1周 | 部门职责、岗位职责 |
| 导师带教 | 入职1-3个月 | 一对一导师指导 |

## 5. 薪酬福利制度

### 5.1 薪酬结构

#### 5.1.1 基本工资
根据岗位职级确定，按月发放。

#### 5.1.2 绩效工资
根据个人和公司业绩情况，按季度或年度发放。

| 绩效等级 | 绩效系数 | 说明 |
|----------|----------|------|
| A（卓越） | 1.5 | 超出预期，表现卓越 |
| B（优秀） | 1.2 | 超过预期 |
| C（达标） | 1.0 | 达到预期 |
| D（待改进） | 0.8 | 未完全达到预期 |

#### 5.1.3 年终奖金
根据公司年度经营状况和个人绩效表现发放，通常为1-3个月基本工资。

#### 5.1.4 项目奖金
针对特定项目或专项任务设置的奖励。

## 6. 考勤与休假制度

### 6.1 工作时间

#### 标准工时制
- **工作日**: 周一至周五
- **工作时间**: 9:00-18:00（午休12:00-13:00）
- **每周工作时间**: 40小时

#### 弹性工作制
- 核心工作时间: 10:00-16:00
- 弹性上下班时间: 8:00-10:00 / 16:00-20:00
- 适用于部分岗位，需部门负责人批准

#### 特殊工时制
- 项目制岗位可申请综合工时制
- 需要人力资源部审批

### 6.2 考勤管理

#### 6.2.1 考勤方式
- 指纹打卡或人脸识别
- 外出需填写《外出申请单》
- 出差需填写《出差申请单》

#### 6.2.2 迟到早退处理

| 迟到/早退时长 | 处理方式 |
|---------------|----------|
| 30分钟以内 | 口头警告 |
| 30分钟-1小时 | 按事假半天处理 |
| 超过1小时 | 按事假整天处理 |
| 每月累计迟到3次 | 书面警告 |

### 6.3 休假制度

#### 6.3.1 年假
- 按司龄计算，详见"带薪年假"部分
- 年假需提前申请，部门负责人审批
- 年假不可跨年累积，特殊情况需CEO审批

#### 6.3.2 病假
- 凭二级甲等及以上医院证明申请
- 2天以内由部门负责人审批
- 3天以上需由人力资源部审批
- 病假期间工资按80%发放

#### 6.3.3 事假
- 全年累计不超过15天
- 按实际天数扣发工资
- 超过15天需经总经理审批

#### 6.3.4 婚假
- 按法定年龄结婚: 3天
- 晚婚（男25岁/女23岁以上）: 10天
- 需提供结婚证明

#### 6.3.5 产假
- 基础产假: 98天
- 难产增加: 15天
- 多胞胎每多1个增加: 15天
- 怀孕2个月以内流产: 15天
- 怀孕2个月以上流产: 30天
- 男员工陪产假: 10天

#### 6.3.6 丧假
- 父母、配偶、子女: 5天
- 祖父母、外祖父母、兄弟姐妹: 3天
- 需提供关系证明

## 7. 绩效考核制度

### 7.1 考核周期

| 考核类型 | 考核周期 | 评估时间 |
|----------|----------|----------|
| 月度考核 | 每月 | 次月5日前 |
| 季度考核 | 每季度 | 季度末月25日前 |
| 年度考核 | 每年 | 次年1月15日前 |

### 7.2 考核维度

| 维度 | 权重 | 说明 |
|------|------|------|
| 业绩目标达成 | 40% | 关键业绩指标完成情况 |
| 工作能力 | 30% | 专业能力、执行力、创新能力 |
| 工作态度 | 20% | 责任心、协作性、纪律性 |
| 人才培养 | 10% | 导师带教、知识分享 |

### 7.3 考核流程

```
制定目标 → 月/季度跟踪 → 自我评估 → 主管评估 → HR复核 → 结果面谈 → 结果应用
```

### 7.4 考核结果应用

| 考核结果 | 绩效系数 | 奖金影响 | 晋升机会 | 其他影响 |
|----------|----------|----------|----------|----------|
| A（卓越） | 1.5 | 显著增加 | 优先考虑 | 股票期权 |
| B（优秀） | 1.2 | 适当增加 | 重点培养 | 培训机会 |
| C（达标） | 1.0 | 正常发放 | 正常晋升 | - |
| D（待改进） | 0.8 | 减少 | 延迟晋升 | 绩效改进计划 |
| E（不合格） | 0 | 无奖金 | 不予晋升 | 岗位调整或离职 |

## 9. 信息安全与保密

### 9.1 信息安全总则

1. **最小权限原则**: 员工只能访问工作所需的最少信息
2. **责任到人原则**: 每个账号的使用责任由持有人承担
3. **预防为主原则**: 主动防范信息安全风险

### 9.2 保密级别分类

| 保密级别 | 标识 | 范围示例 | 泄露处理 |
|----------|------|----------|----------|
| 绝密 | ★★★ | 核心技术、战略规划、财务数据 | 立即开除并追究法律责任 |
| 机密 | ★★ | 重要客户信息、薪酬数据、内部文件 | 降职或开除 |
| 秘密 | ★ | 一般业务数据、项目资料 | 警告或绩效扣减 |
| 内部 | ○ | 公司内部公开信息 | 口头警告或通报批评 |

### 9.3 信息安全规范

#### 9.3.1 账户与密码
- 账户不得借给他人使用
- 密码必须包含大小写字母、数字和特殊字符
- 密码每90天更换一次
- 不得将密码写在纸上或存储在不安全的地方

#### 9.3.2 计算机与网络
- 离开工位时必须锁屏（Win+L）
- 不得使用非公司设备访问公司网络
- 不得安装未经许可的软件
- 定期更新操作系统和杀毒软件

#### 9.3.3 数据保护
- 敏感数据必须加密存储
- 文件传输使用公司指定的安全工具
- 禁止使用个人邮箱传输公司敏感信息
- 废弃文件必须彻底销毁

### 9.4 保密协议

所有员工入职时必须签署《保密协议》，明确保密义务和违约责任。保密协议在离职后仍然有效，具体期限根据岗位级别确定。

- 女士化妆应淡雅得体
- 保持口气清新，身上无异味
- 指甲修剪整齐，女性美甲颜色不宜过艳

### 10.2 办公礼仪

#### 10.2.1 语言规范

- 准时参加会议，迟到者主动道歉

- 积极参与讨论，发言简明扼要
- 会议记录及时整理并发送

### 11.2 员工申诉机制

员工如认为自身权益受到侵害，可通过以下途径申诉：

1. **直接沟通**: 与当事人或直属上级沟通解决
2. **部门协调**: 请求部门HRBP协助调解
3. **正式申诉**: 向人力资源部提交书面申诉
4. **高层反馈**: 通过CEO邮箱直接反馈

### 11.3 员工关怀

#### 11.3.1 心理健康支持
- 提供免费心理咨询服务

#### 11.3.2 困难员工帮扶
- 设立员工互助基金
- 提供紧急救助通道
- 组织志愿者帮扶活动

### 11.4 劳动争议处理

如发生劳动争议，双方应首先通过友好协商解决。协商不成的，可向劳动争议调解委员会申请调解，或向劳动仲裁部门申请仲裁。

## 12. 离职流程

### 12.1 离职类型

| 离职类型 | 说明 | 通知期 |
|----------|------|--------|
| 主动辞职 | 员工主动提出离职 | 30天（试用期为3天） |
| 协商离职 | 双方协商一致解除 | 协商确定 |
| 合同到期不续签 | 劳动合同到期 | 公司提前30天通知 |
| 违纪辞退 | 严重违反公司规定 | 无 |
| 裁员 | 公司经营需要 | 按照N+1标准 |

### 12.2 离职流程

```
提交辞职申请 → 部门负责人审批 → HR审批 → 完成工作交接 → 归还公司资产 → 结算工资 → 办理离职证明 → 社保转移
```

### 12.3 离职交接

离职员工需完成以下交接工作：

1. **工作文件**: 整理并移交所有工作文档
2. **项目进展**: 详细说明项目当前状态和后续计划
3. **资产归还**: 归还笔记本电脑、工牌、门禁卡等
4. **账号权限**: 配合IT部门注销各类系统账号
5. **财务结算**: 结清借款、报销等财务事项

### 12.4 离职后义务

离职后员工仍需履行以下义务：

- **保密义务**: 继续保守公司商业秘密
- **竞业限制**: 部分核心岗位需遵守竞业限制协议
- **回访配合**: 如有需要，配合公司进行背景调查
